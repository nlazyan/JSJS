Եթե Ձեզ դուր է գալիս JavaScrip պարզեցված ժառգոնը, ստուգեք կապակցված պրոյեկտները:

A
  • Ajax (անգլ. Asynchronous Javascript and XML - «Ասինխրոն Javascript և XML) - տեխնոլոգիա ասինխրոն HTTP հարցումների համար
  • AMD (անգլ. Asynchronous Module Definition - «Մոդուլի անինխրոն սահմանում») - ստանդարտ, որը սահմանում է(բնութագրում է)՝ ինչպես JavaScript գրադարանները և մոդուլները բեռնել ասինխրոն
  • AngularJS - դինամիկ վեբ հավելվածների կառուցվածքային հարթակ(ֆրեմվորկ)
  • Apache Cordova - Մոբիլ հավելվածների մշակման հայտնի հարթակ է, որը, ի սկզբանե, ստեղծվել է Nitobi-ի կողմից
  • Arity - ֆունկցիայի արգումենտների քանակն է
B
  • Babel - JavaScript վերափոխման գործիքակազմ, որը ի սկզբանե եղել է որպես ECMAScript 2015/ES6 կոդի թարգմանիչ (transpiler - տրանսպիլեռ)
  • Backbone - դինամիկ վեբ ափերի համար կառուցվածքային հարթակ(ֆրեյմորկ)
  • BEM - Յանդեքսի կողմից մշակված և կիրառվող գրադարաններ և մեթոդոլոգիաներ յուզեր(օգտագործողի) ինտերֆեյսի կառուցման համար
  • Bluebird - լիարժեք ֆունկցիոնալ գրադարան փրոմիսների(Promise) հետ աշխատելու համար ժամանակակից հնարավորությունների և արտադրողականության վրա կենտրոնացմամբ
  • Bower - ֆրոնթէնդ-կախվածությունների համար փաթեթների մենեջեր
  • Broccoli - արագ և ապահով ֆայլալար(անգլ. asset pipeline)
  • Browserify - գործիք, որը հնարավոր է դարձնում բրոուզերում Node.js-ից `require` ֆունկցիաների կիրառումը
  • Brunch - գործիք պռոյեկտի սկզբնական կոդից տեղակայման պատրաստ(deployment-ready) ֆայլերի արտադրության համար
C
  • Canvas - HTML - էլեմենտ 2D կամ 3D գրաֆիկական հավելվածներ ստեղծելու համար
  • Chai - assertion-գրադարան, որը օգտագործվում է JavaScript տեստավորման ֆրեմվորկների հետ
  • Chakra - JavaScript շարժիչ, որը մշակվել է Microsoft-ի կողմից իր Edge բրոուզերի համար, որը նաև կարող է օգտագործվել Node.js-ի հետ V8-ի փոխարեն
  • Closure - ֆունկցիա, որը ներդրված է այլ ֆունկցիաների մեջ և ունի հասանելիություն արտաքին ֆունկցիաների փոփոխականներին, միևնույն ժամանակ նրա բոլոր ներքին փոփոխականները թաքնված են արտաքին ֆունկցիաների համար
  • Closure Compiler - JavaScript ստուգիչ(checker) և օպտիմալացման միջոց
  • Coffeescript - մետալեզու, որը կոմպիլացվում է JavaScript-ում
  • CommonJS - պրոյեկտ, որի նպատակն է JavaScript- ի էկոհամակարգը դիտարկել բրաուզերից դուրս(օրինակ՝ սերվերի վրա կամ մայրենի դեսկթոփ հավելվածների(native desktop applications) համաչ)
  • CORS (ռեսուրսների միջդոմենային փոխանակում)- տեխնոլոգիա, որը թույլ է տալիս վեբ-էջին հասանելիություն տալ այլ դոմեյնի ռեսուրսներին
  • CouchDB - NoSQL տվյալների բազա, որը օգտագործում է JavaScript-ը որպես հարցումների լեզու և HTTP-ն՝ որպես API
  • Currying - պրոցես, որը մի քանի պարամետրերով(արգումենտներով) ֆունկցիան ձևափոխում է մեկ պարամետր ունեցող ֆունկցիաների շղթայի
